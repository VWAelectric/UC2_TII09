Introdução à Tag < img >

A tag < img > em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja, não possui uma tag de fechamento e contém apenas atributos que definem o conteúdo.

Atributos principais:

  • src(source): define o caminho da imagem que será exibida. pode ser uma URL externa ou um caminho relativo dentro do servidor.

    Exemplo 1:

    < img src="../09/img/imagem_1.jpg" alt="Imagem de montanha" >

    Resultado:

  • alt (alternative text): fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada. ele também melhora a acessibilidade para leitores de tela.

    Exemplo 2:

    < img src="../09/img/gato andando.gif" alt="Giff gato andando" >

    Resultado:

  • src também pode ser de um link externo:

    Exemplo 3:

    < img src="https://images.pexels.com/photos/1366919/pexels-photo-1366919.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1" alt="Montanha Congelada" >

Tipos comuns de imagens

Os formatos de imagens mais usados na web são:

  • (.jpg ou .jpeg): Formato utilizando amplamente para fotografias e imagens com muitos detalhes e cores. Possui compressão com perda de qualidade (lossy).
  • (.png): Usado para imagens com fundo transparente ou que necessitam de alta qualidade, sem perda de detalhes. Utiliza compressão sem perda (lossless).
  • GIF (.gif): Formato ideal para imagens animadas e pequenos ícones. Suporta transparência, mas tem um número limitado de cores.
