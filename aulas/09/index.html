Introdução à Tag < img >

A tag < img > em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja, não possui uma tag de fechamento e contém apenas atributos que definem o conteúdo.

Atributos principais:

  • src(source): define o caminho da imagem que será exibida. pode ser uma URL externa ou um caminho relativo dentro do servidor.
